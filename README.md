# brunch-starter-kit

> Starter Kit based on [Brunch](https://brunch.io/) for work with ES6, Autoprefixer and SASS

## Setup

```bash
# install dependencies
npm install

# start auto reload server
npm start

# build for production
npm run build
```

## Folder structure

    .
    ├── dist                        # Compiled files
    └── src                         # Source files

         │   ├── images
         │   ├── index.html
         │   ├── robot.txt
         │   └── site.webmanifest
         ├── scripts
         │   └── app.js
         └── styles
             └── app.scss
